**Helper functions to add at module level:**

1. `checkPaused(cardId: string): Promise<boolean>` — re-reads `prisma.card.findUnique({ where: { id: cardId }, select: { pipelineStatus: true } })` and returns `card?.pipelineStatus === 'PAUSED'`. MUST always query DB — never cache.

3. `getNextPosition(columnId: string): Promise<number>` — finds last card in target column, returns `(lastPosition ?? 0) + 1000`.

**Main function flow:**

1. **Load card** with select: `{ id, title, description, pipelineStatus, columnId, cardType }`

2. **Validate state**: card must be `QUEUED`. If not, throw.

3. **Determine resume point** (retry from failed stage):
   - Find the latest PipelineRun for this card: `prisma.pipelineRun.findFirst({ where: { cardId }, orderBy: { createdAt: 'desc' } })`
   - If `latestRun?.status === 'FAILED'`:
     - If `latestRun.stage === 'PLANNING'` -> start from PLANNING (default)
     - If `latestRun.stage === 'EXECUTING'` -> skip planning, load planText from previous run's PipelineMessage where `artifactType === 'plan'`, start from EXECUTING
     - If `latestRun.stage === 'TESTING'` -> skip planning and execution, load planText and execResult from previous run's messages, start from TESTING
   - If no failed run or `latestRun.stage === 'PLANNING'` -> start from beginning
   - Set `startStage` variable: `'PLANNING' | 'EXECUTING' | 'TESTING'`

4. **Load all 5 column IDs** at startup: `prisma.column.findMany({ select: { id, name } })`. Map by name for lookups: `const columnMap = new Map(columns.map(c => [c.name, c.id]))`. Validate that Suunnittelu, Toteutus, Testaus, and Valmis columns exist.

5. **Create new PipelineRun** with `stage: startStage, status: startStage` (e.g., `'PLANNING'` or `'EXECUTING'`).

6. **Stage 1: PLANNING** (skip if startStage is EXECUTING or TESTING):
   - Move card to Suunnittelu column with PLANNING status using `advanceToStage`
   - Store user prompt as PipelineMessage (role: 'user')
   - Call `runPlanningStage({ title: card.title, description: card.description })`
   - Store result as PipelineMessage (role: 'assistant', artifactType: 'plan')
   - Set `planText = result`

7. **Pause check 1** (between planning and execution):
   - Call `checkPaused(cardId)`. If true:
     - Update PipelineRun status to PAUSED
     - Update Card pipelineStatus to PAUSED (it keeps its current column position)
     - Return (exit worker cleanly)

8. **Stage 2: EXECUTING** (skip if startStage is TESTING):
   - Move card to Toteutus column with EXECUTING status using `advanceToStage`
   - Branch on `card.cardType`:
     - If `'CODE'`: call `ensureWorkspace(cardId)` then `runExecutionStage({ id: card.id, title: card.title }, planText!, workspacePath)`
     - Otherwise: call `runExecutionStageApi({ title: card.title }, planText!)`
   - Store result as PipelineMessage (role: 'assistant', artifactType: `card.cardType === 'CODE' ? 'code' : 'execution'`)
   - Set `execResult = result`

9. **Pause check 2** (between execution and testing):
   - Same pattern as pause check 1.

10. **Stage 3: TESTING**:
    - Move card to Testaus column with TESTING status using `advanceToStage`
    - Call `runTestingStage({ title: card.title }, planText!, execResult!)`
    - Store result as PipelineMessage (role: 'assistant', artifactType: 'test_report')

11. **Complete**: Move card to Valmis column with COMPLETED status using `advanceToStage`. Update PipelineRun status to COMPLETED.

**Error handling:** Keep existing try/catch pattern — on error, set card to FAILED, update PipelineRun with error message. The `finally` block disconnects Prisma and exits.

**A. Add `pausePipeline` Server Action:**
```typescript
export async function pausePipeline(
  input: { cardId: string }
): Promise<ActionResult<void>> {
  const parsed = pausePipelineSchema.safeParse(input)
  if (!parsed.success) {
    return { success: false, error: formatZodError(parsed.error) }
  }

  const { cardId } = parsed.data

  try {
    const card = await prisma.card.findUnique({
      where: { id: cardId },
      select: { pipelineStatus: true },
    })

    const pauseable: PipelineStatus[] = ['PLANNING', 'EXECUTING', 'TESTING']
    if (!card || !pauseable.includes(card.pipelineStatus as PipelineStatus)) {
      return { success: false, error: 'Pipeline ei ole käynnissä' }
    }

    await prisma.card.update({
      where: { id: cardId },
      data: { pipelineStatus: 'PAUSED' },
    })

    revalidatePath('/')
    return { success: true, data: undefined }
  } catch {
    return { success: false, error: 'Pipelinen pysäytys epäonnistui' }
  }
}
```
Import `pausePipelineSchema` from `'./ai-schemas'` and `PipelineStatus` from `'@/types'`.
