<task type="auto">
  <name>Task 2: Implement execution and testing stage runners in pipeline-stages.ts</name>
  <files>src/workers/pipeline-stages.ts</files>
  <action>
Extend `src/workers/pipeline-stages.ts` — keep existing `runPlanningStage` unchanged, add three new exports:

1. **`runExecutionStage(card, planText, workspacePath): Promise<string>`** — For CODE cards. Uses `@anthropic-ai/claude-agent-sdk` `query()`:
   - Import `{ query }` from `'@anthropic-ai/claude-agent-sdk'`
   - Call `query({ prompt: buildExecutionPrompt(card.title, planText), options: { cwd: workspacePath, permissionMode: 'bypassPermissions', allowDangerouslySkipPermissions: true, allowedTools: ['Read', 'Write', 'Edit', 'Bash', 'Glob', 'Grep'], maxTurns: 30, env: { ANTHROPIC_API_KEY: process.env.ANTHROPIC_API_KEY!, PATH: process.env.PATH!, HOME: process.env.HOME! } } })`
   - Iterate with `for await (const message of query(...))` — collect text from `message.type === 'assistant'` messages where content blocks have `'text' in block`
   - Check for `message.type === 'result'` and throw if `message.subtype !== 'success'`
   - CRITICAL: Never `break` or `return` inside the for-await loop — consume all messages to prevent orphan processes
   - Return collected text joined with `'\n\n'`, fallback to `'Suoritus valmis (ei tekstituotosta)'`
   - Type the card parameter as `{ id: string; title: string }`

2. **`runExecutionStageApi(card, planText): Promise<string>`** — For RESEARCH/BUSINESS/GENERAL cards. Uses Anthropic `messages.create` (same pattern as existing `runPlanningStage`):
   - Create fresh `new Anthropic()` instance
   - Call `messages.create` with `PIPELINE_MODEL`, `max_tokens: 8096`, single user message with `buildExecutionPromptApi(card.title, planText)`
   - Extract text block, throw if missing: `'Ei tekstisisältöä API-vastauksessa'`
   - Type the card parameter as `{ title: string }`

3. **`runTestingStage(card, planText, executionResult): Promise<string>`** — For ALL card types. Uses Anthropic `messages.create`:
   - Create fresh `new Anthropic()` instance
   - Call `messages.create` with `PIPELINE_MODEL`, `max_tokens: 4096`, single user message with `buildTestingPrompt(card.title, planText, executionResult)`
   - Extract text block, throw if missing: `'Ei tekstisisältöä testausvastauksessa'`
   - Type the card parameter as `{ title: string }`

Import the new prompt builders from `./prompts`. Keep PIPELINE_MODEL constant at module level (already exists).
  </action>
