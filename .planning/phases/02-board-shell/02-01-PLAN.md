**page.tsx** — Replace the entire default scaffold:
- This is an async Server Component (no `'use client'` directive).
- Import `dynamic` from `next/dynamic`.
- Import `getBoard` from `@/actions/tasks`.
- Import `ColumnWithCards` type from `@/types` and `SerializedColumn` type from `@/types`.
- Create `serializeColumn(col: ColumnWithCards): SerializedColumn` function that:
  - Spreads `col` and overrides `createdAt` with `col.createdAt.toISOString()`.
  - Maps `col.cards` to override `dueDate` (`.toISOString()` or null), `createdAt` (`.toISOString()`), `updatedAt` (`.toISOString()`).
  - This is critical — without it, React 19 throws "Only plain objects can be passed to Client Components" because Prisma returns `Date` objects.
- Create `const Board = dynamic(() => import('@/components/board/Board'), { ssr: false })`.
  - `ssr: false` prevents server rendering of the Board tree. This avoids hydration mismatches from `new Date()` in DateDisplay AND pre-positions the architecture for Phase 4 DnD (which requires browser-only APIs).
  - Do NOT add a `loading` prop to `dynamic()` — `app/loading.tsx` handles the fallback.
- Export default async function `HomePage()`:
  - `const columns = await getBoard()`
  - `const serialized = columns.map(serializeColumn)`
  - Return `<main className="h-screen overflow-hidden"><Board columns={serialized} /></main>`.
