**createTask implementation:**
- Import `positionAfterLast` from `@/lib/positions`
- Import `revalidatePath` from `next/cache`
- After Zod parse succeeds, query for the last card in the target column: `prisma.card.findFirst({ where: { columnId: parsed.data.columnId }, orderBy: { position: 'desc' }, select: { position: true } })`
- Calculate position with `positionAfterLast(last?.position ?? 0)`
- Create card with `prisma.card.create()` — data fields: title, description (default ''), priority (default 'MEDIUM'), dueDate (convert string to `new Date()` or null), position, columnId. If labelIds are provided, use nested `labels: { create: labelIds.map(id => ({ label: { connect: { id } } })) }`. Include: `{ labels: { include: { label: true } } }`
- Call `revalidatePath('/')`
- Return `{ success: true, data: card }`
- Wrap the Prisma call in try/catch — on error return `{ success: false, error: 'Kortin luonti epaonnistui' }`

**updateTask implementation:**
- After Zod parse, destructure: `const { id, labelIds, dueDate, ...rest } = parsed.data`
- Use `prisma.card.update({ where: { id }, data: { ...rest, dueDate: dueDate === undefined ? undefined : dueDate ? new Date(dueDate) : null, ...(labelIds !== undefined && { labels: { deleteMany: {}, create: labelIds.map(labelId => ({ label: { connect: { id: labelId } } })) } }) }, include: { labels: { include: { label: true } } } })`
- CRITICAL: Use `deleteMany: {}` + `create` for labels, NOT `set` — CardLabel is an explicit M:N join table
- Call `revalidatePath('/')`
- Return `{ success: true, data: card }`
- Wrap in try/catch — on error return `{ success: false, error: 'Kortin paivitys epaonnistui' }`

**deleteTask implementation:**
- Keep the existing id validation
- Call `prisma.card.delete({ where: { id } })` — CardLabel rows auto-cascade via onDelete: Cascade
- Call `revalidatePath('/')`
- Return `{ success: true, data: undefined }`
- Wrap in try/catch — on error return `{ success: false, error: 'Kortin poisto epaonnistui' }`
