  <name>Task 1: VPS initial setup and first deploy</name>
  <files>VPS remote files only -- no local files modified</files>
  <action>
SSH into the Hostinger VPS and run the following commands. Replace `yourdomain.com` with your actual domain. Ensure DNS A record points to VPS IP before step 9 (Certbot).

**Step 1: Create deploy user and directories (as root)**
```bash
adduser deploy
usermod -aG sudo deploy

sudo mkdir -p /var/data/taskboard
sudo chown deploy:deploy /var/data/taskboard
sudo mkdir -p /var/log/taskboard
sudo chown deploy:deploy /var/log/taskboard
```

**Step 2: Install Node.js via NVM (as deploy user)**
```bash
su - deploy
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install 22
nvm use 22
nvm alias default 22
node --version  # Should show v22.x.x
```

**Step 3: Install PM2 globally**
```bash
npm install -g pm2
```

**Step 4: Clone repo and configure**
```bash
git clone https://github.com/YOUR_USERNAME/taskboard.git /home/deploy/taskboard
cd /home/deploy/taskboard
cp .env.example .env.production
nano .env.production
# Fill in real ANTHROPIC_API_KEY
# Verify DATABASE_URL is file:/var/data/taskboard/tasks.db
```

**Step 5: First deploy**
```bash
cd /home/deploy/taskboard
npm ci --production=false
npx prisma generate
DATABASE_URL="file:/var/data/taskboard/tasks.db" npx prisma migrate deploy
DATABASE_URL="file:/var/data/taskboard/tasks.db" npx tsx prisma/seed.ts
npm run build
cp -r public .next/standalone/
cp -r .next/static .next/standalone/.next/
pm2 start ecosystem.config.js
pm2 save
```

**Step 6: Configure PM2 startup on boot**
```bash
pm2 startup
# PM2 prints a command -- copy and run that EXACT command as root
pm2 save
```

**Step 7: Install and configure Nginx**
```bash
sudo apt update && sudo apt install -y nginx
sudo cp /home/deploy/taskboard/nginx/taskboard.conf /etc/nginx/sites-available/taskboard
sudo nano /etc/nginx/sites-available/taskboard  # Replace yourdomain.com
sudo ln -s /etc/nginx/sites-available/taskboard /etc/nginx/sites-enabled/
sudo rm -f /etc/nginx/sites-enabled/default
sudo nginx -t
sudo systemctl reload nginx
```

**Step 8: Configure UFW firewall**
```bash
sudo ufw allow ssh
sudo ufw allow 'Nginx Full'
sudo ufw enable
sudo ufw status  # Verify 3000 is NOT listed
```

**Step 9: Install SSL with Certbot**
```bash
sudo snap install --classic certbot
sudo ln -s /snap/bin/certbot /usr/bin/certbot
sudo certbot --nginx -d yourdomain.com -d www.yourdomain.com
sudo certbot renew --dry-run
```

**Step 10: Quick verify**
```bash
pm2 list
curl -I http://localhost:3000
sudo ufw status
```
